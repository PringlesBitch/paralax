# Vibecode Landing Page

A modern, responsive landing page for Vibecode - an AI-powered mobile app builder that lets users create apps without coding.

## Features

- Modern, responsive design
- Smooth animations and transitions
- Mobile-friendly navigation
- Beautiful gradient effects
- Interactive elements

## Running Locally

### Option 1: Python HTTP Server

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Option 2: Node.js HTTP Server

If you have Node.js installed:

```bash
npx http-server -p 8000
```

Then open http://localhost:8000 in your browser.

### Option 3: VS Code Live Server

If you're using VS Code, install the "Live Server" extension and click "Go Live" in the status bar.

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
